About the program

Über das Programm

Amazon Web Services (AWS) re/Start is a free skills development program that prepares you for entry-level careers in the cloud. The program’s mission is to build local talent by providing AWS Cloud skills development and job opportunities. AWS re/Start is collaborating with Primed Talent GmbH, a talent transformation organization for this program. Together, AWS re/Start and Primed Talent GmbH are excited to help you launch a career in cloud computing.

Amazon Web Services (AWS) re/Start ist ein kostenloses Programm zur Kompetenzentwicklung, das Sie auf Einstiegskarrieren in der Cloud vorbereitet.Die Mission des Programms besteht darin, lokale Talente aufzubauen, indem AWS Cloud-Fähigkeiten und Beschäftigungsmöglichkeiten bereitgestellt werden. AWS re/Start arbeitet für dieses Programm mit der Primed Talent GmbH zusammen. AWS re/Start und Primed Talent GmbH freuen sich, Sie beim Start in eine Karriere im Bereich Cloud Computing zu unterstützen.

The 12-week program featuring real-world scenario-based learning, hands-on labs, and coursework is supported by professional mentors and accredited trainers. The program also provides structured interventions that help learners with resume and interview coaching to prepare for employer meetings and interviews to launch an entry-level cloud career.

Das 12-wöchige Programm mit realem szenariobasiertem Lernen, praktischen Übungen und Kursarbeit wird von professionellen Mentoren und akkreditierten Trainern unterstützt. Das Programm bietet auch strukturierte Interventionen, die den Lernenden mit Lebenslauf- und Interview-Coaching helfen, sich auf Arbeitgebertreffen und Interviews vorzubereiten, um eine Einstiegs-Cloud-Karriere zu starten.

What skills will you learn during the 12 weeks?

Welche Fähigkeiten wirst du in den 12 Wochen erlernen?

Week 1 – Week 7

Woche 1 – Woche 7

Focus on Core IT areas for a Support / Operations such as Linux, Networking, Security, Python Programming and Database

Fokus auf IT-Kernbereiche für einen Support / Operations wie Linux, Networking, Security, Python-Programmierung und Datenbank

Week 7 – Week 9

Woche 7 – Woche 9

Focus on AWS core services such as Cloud Economics, Global Infrastructure, Elastic Load Balancing, Amazon CloudWatch, Auto Scaling, Cloud Architecting & Security

Konzentrieren Sie sich auf AWS-Kernservices wie Cloud Economics, Global Infrastructure, Elastic Load Balancing, Amazon CloudWatch, Auto Scaling, Cloud Architecting & Security

Week 9 – Week 12

Woche 9 – Woche 12

Deep dive into AWS services from an operational viewpoint such as Tooling-and-Automation, Computing-Servers, Scaling-and-Name-Resolution, Containers-and-Serverless, Networking, Monitoring-and-Security

Vertiefender Einblick in AWS-Services aus betrieblicher Sicht wie Tooling-and-Automation, Computing-Server, Scaling-and-Name-Resolution, Containers-and-Serverless, Networking, Monitoring-and-Security
