AWS re/Start

Amazon Web Services (AWS) re/Start is a free skills development program that prepares learners for entry-level careers in the cloud.

The program’s mission is to build local talent by providing AWS Cloud skills development and job opportunities. Together, AWS re/Start and Primed Talent GmbH are excited to help you launch a career in cloud computing.

Amazon Web Services (AWS) re/Start ist eine kostenlose Kompetenzentwicklung Programm, das die Lernenden auf den Berufseinstieg im Bereich vorbereitet Wolke.

Die Mission des Programms besteht darin, lokale Talente aufzubauen, indem AWS Cloud-Fähigkeiten und Stellenangebote angeboten werden. AWS re/Start und Primed Talent GmbH freuen sich, Sie beim Start in eine Karriere im Bereich Cloud Computing zu unterstützen.
